# Threadbound

> *Echoes shape the future. Reflections reveal the truth.*

## Overview

Threadbound is a narrative puzzle-adventure game created for a Game Jam based on the theme **"Echoes and Reflections."**

The game explores how every action leaves a lasting impact. Instead of immediately rewarding or punishing the player, choices are stored within the world and later return as echoes that influence future gameplay. The world becomes a reflection of the player's own decisions, encouraging empathy, trust, and personal growth.

Rather than relying on dialogue to explain its themes, Threadbound communicates its story through gameplay mechanics.

---

## Core Gameplay

### Echo System

Every meaningful decision is remembered by the world.

Actions made earlier in the game return later as:


- Consequences
- Environmental changes
- Different interactions with characters

The player learns that every choice matters.

### Reflection Mechanic

Players eventually encounter an echo of their past self trapped in an endless cycle of repeating the same mistake.

Progress is achieved by choosing differently rather than erasing the past.

### Bond Mechanic

Strength comes from connection.

Players are mechanically stronger when working together than when facing challenges alone.

Inspired by the idea that a cord of three strands is stronger than one.

### Trust Mechanic

The player encounters someone who resembles a person who previously caused harm.

The game rewards the player for judging the new character by their own actions instead of assuming history will repeat itself.

## Getting Started

### Prerequisites

Make sure you have installed:

- Java 17 (or the version required by this project)
- Apache Maven

Verify installation:

```bash
java -version
mvn -version
```

---

## Running the Project

Clone the repository:

```bash
git clone <https://github.com/Mapaseka-Moloi/Thread-Bound>
```

Navigate into the project:

```bash
cd Threadbound
```

Run the application:

```bash
mvn javafx:run
```
